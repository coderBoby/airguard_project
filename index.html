System Architecture

How AirGuard Works

A seamlessly integrated hardware–software pipeline that reacts in real time — from detection to deployment in under a second.

01
📡

Detect

Ultrasonic (HC-SR04) & PIR sensors on the rooftop continuously monitor for any object in free-fall trajectory.

02
🧠

Analyse

Arduino Uno / ESP32 microcontroller processes sensor fusion data using fall-pattern algorithms to confirm a human fall.

03
⚡

Trigger

A 5V relay module fires an electromagnetic signal to the deployment mechanism in milliseconds of confirmation.

04
🛡️

Deploy

A DC motor-driven cloth-net safety cushion unfurls instantly, absorbing impact energy and preventing fatal injury.

Hardware Components

Built From The Ground Up

Every component carefully chosen for speed, reliability, and real-world feasibility.

🔌

Arduino Uno R3

The brain of our system. Processes sensor data in real-time and fires trigger signals using embedded C firmware.

ATmega328P
📡

HC-SR04 Ultrasonic Sensor

Dual sensors for multi-angle fall detection. Measures distance changes at high frequency for accurate trajectory analysis.

×2 Units
⚡

5V Relay Module (1-channel)

Serves as the activation switch — triggers the motor deployment mechanism with a clean, low-latency signal.

1-Channel
🔧

DC Motor (5V–12V)

Drives the cloth-net deployment mechanism. Compact yet powerful enough to unfurl the safety net rapidly.

5V–12V
🎛️

L298N Motor Driver

Controls the DC motor speed and direction. Enables controlled, reliable deployment every single time.

Dual H-Bridge
🔔

Buzzer + LED Alert

Active buzzer and status LEDs provide immediate local alerts upon fall detection and system activation.

Technical Specifications

Built With Precision

🔌

Microcontroller

Arduino Uno R3 at the core — handling real-time sensor processing, decision logic, and relay triggering with sub-millisecond response.

ATmega328P
📡

Sensors

HC-SR04 Ultrasonic sensors for multi-layered fall-pattern recognition and false-positive rejection using dual-sensor fusion logic.

Dual-Sensor Fusion

Deployment Net
